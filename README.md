# Recetario Peruano - Proyecto HTML

Este es un proyecto sencillo hecho únicamente con **HTML**, en el cual se muestran algunas recetas tradicionales de la gastronomía peruana.

## Estructura del proyecto

- **index.html** → Página principal que contiene enlaces a cada receta.
- **/recetas/** → Carpeta donde se encuentran las páginas HTML de cada receta:
  - causa.html
  - salchipapa.html
  - arroz_con_leche.html
- **/images/** → Carpeta donde están guardadas las imágenes utilizadas en las páginas.

## Recetas incluidas

### Causa Limeña
Plato tradicional hecho a base de papa amarilla, relleno de pollo, atún u otros ingredientes, servido frío y decorado con aceitunas, huevo duro y palta.

### Salchipapa
Plato popular de comida rápida peruana, hecho con papas fritas y salchichas, acompañado de distintas salsas.

### Arroz con Leche
Postre cremoso y dulce, elaborado con arroz, leche, canela y azúcar, muy consumido en Perú.

## Cómo usar el proyecto

1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador.
3. Haz clic en los enlaces para ver cada receta.

Este proyecto está hecho únicamente con HTML para fines educativos mientras aprendo desarrollo web.
